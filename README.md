# Pheasant.js

**Pheasant.js** makes colors handling easy.

## Color Formats Support

| Format      | Parsing | Stringifying |
|-------------|:-------:|:------------:|
| color names | ✔       | ✔            |
| hsl()       | ✔       | ✗            |
| hsla()      | ✔       | ✗            |
| rgb()       | ✔       | ✔            |
| rgba()      | ✔       | ✔            |
